# Emotion-Recognition-and-Content-Recommendation-System-using-deep learning

## Project Description
This project is an AI-based application that detects human emotions from facial expressions and recommends suitable content based on the detected emotion. The system uses machine learning and computer vision techniques for emotion analysis.

## Features
- Real-time emotion detection
- Face recognition using webcam
- Emotion classification
- Personalized content recommendation
- Interactive user interface

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy
- Flask

## Emotions Detected
- Happy
- Sad
- Angry
- Neutral
- Surprise

## How It Works
1. Capture image/video using webcam
2. Detect face from the input
3. Analyze facial expression
4. Predict emotion using trained model
5. Recommend suitable content

## Applications
- Entertainment platforms
- Mental wellness support
- Personalized recommendation systems
- Human-computer interaction
